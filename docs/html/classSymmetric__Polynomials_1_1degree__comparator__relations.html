Detailed Description

The comparator that checks if the degree of a given powers vector is equal to a desired degree, and further disqualifies a powers vector if it contains a relation.

Eg a relation can be \(a_1^2a_2=\cdots\). In that case, a power vector [2,1,0,...,] is disqualified, and so are all [v_1,v_2,v_3,...] with v_1>=2 and v_2>=1 since these also contain the relation The relations are allowed to be very complicated (think equivariant Chern classes) hence why they are not encoded in a rel_t
