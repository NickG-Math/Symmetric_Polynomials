32 \section sym Symmetric polynomials
33 
34 Let \f$R=\mathbf Z[x_1,...,x_n]\f$; there is an obvious action on \f$R\f$ by the symmetric group \f$\Sigma_n\f$ and the fixed points \f$R^{\Sigma_n}\f$ i.e. the symmetric polynomials, form a polynomial algebra on the elementary symmetric polynomials:
35 \f[R^{\Sigma_n}=\mathbf Z[\sigma_1,...,\sigma_n]\f] where
36 \f[\sigma_k(x_1,...,x_n)=\sum_{1\le i_1< \cdots< i_k\le n}x_{i_1}....x_{i_k}\f]
37 Furthermore, there is a simple algorithm for writing every symmetric polynomial on the \f$x_i\f$ as a polynomial on the \f$\sigma_i\f$. This library implements that algorithm.

40 \section hir Symmetric polynomials with 'half idempotent' relations
41 
42 Let \f$R=\mathbf Z[x_1,...,x_n,y_1,...,y_n]/(y_i^2=y_i)\f$; there is an obvious action on \f$R\f$ by the symmetric group \f$\Sigma_n\f$ permuting the \f$x_i\f$ and \f$y_i\f$ variables separately. A minimal description of the \f$R^{\Sigma_n}\f$ is now more difficult:
43 \f[R^{\Sigma_n}=\frac{\mathbf Z[\gamma_{s,i}]}{\gamma_{s,i}\gamma_{t,j}=r_{n,s,i,t,j}\gamma_{t,0}\gamma_{s,\min(i+j,n)}+\cdots}\f] where the "twisted Chern classes" are:
44 \f[\gamma_{s,i}=
45 \sum_{1\le j_1< \cdots< j_s\le n, 1\le k_1< \cdots< k_i\le n\\ j_u\neq k_v}
46 x_{j_1}....x_{j_s}y_{k_1}\cdots y_{k_i}\f]
47 for \f$0\le s\le n\f$ and \f$0\le i\le n-s\f$. The coefficient \f$r_{n,s,i,t,j}\f$ in the relations is given by the binomial:
48 \f[{\min(i+j+s,n)-t}\choose {j}\f]
49 Moreover, the relations require the indices \f$s,i,t,j\f$ to satisfy \f$s\le t\le s+i\f$ and \f$i,j>0\f$.
50 
51 For convenience we set:
52 \f[\alpha_i=\gamma_{0,i}=\sigma_i(y_1,...,y_n)\f]
53 \f[c_s=\gamma_{s,0}=\sigma_s(x_1,...,x_n)\f]
54 Over \f$\mathbf Q\f$, the \f$\alpha_1^i\f$ can generate all \f$\alpha_i\f$ via:
55 \f[\alpha_i=\frac{\alpha(\alpha-1)\cdots(\alpha-i+1)}{i!}\f]
56 however for speed+numerical stability we prefer to use all \f$\alpha_i\f$ and have \f$\mathbf Z\f$ coefficients in our relations.
57 
58 This library implements an algorithm that can write every element in \f$R^{\Sigma_n}\f$ as a polynomial on the generators \f$\alpha_i,c_i,\gamma_{s,i}\f$ and further produce every relation explicitly.
59 
60 The "twisted Pontryagin/symplectic classes" are defined as:
61 \f[\kappa_{s,i}=\sum_{1\le j_1< \cdots< j_s\le n,
62 1\le k_1< \cdots< k_i\le n\\
63 j_u\neq k_v}x^2_{j_1}....x^2_{j_s}y_{k_1}\cdots y_{k_i}\f]
64 This library also allows one to write the \f$\kappa_{s,i}\f$ in terms of the \f$\gamma_{s,i}\f$

85 \subsection mono Polynomials
86 
87 A polynomial \f$p\f$ in the graded ring \f$\mathbf Z[x_1,...,x_n]\f$, \f$|x_i|=1\f$, can be declared as:
88 
89  Poly<int, StandardVariables<>> p;
90 
91 Here:
92 - ```int``` is the type of the scalar coefficients (i.e. elements of the base ring which in this case is \f$\mathbf Z\f$), so it can be replaced by ```double``` and even
93 user-defined scalar types.
94 - \ref symmp::StandardVariables specifies variables in degree \f$1\f$, with names \f$x_i\f$ and no relations. We shall explain alternative variable configurations in the next few subsections.
95 
96 To insert a monomial \f$cx_1^{a_1}\cdots x_n^{a_n}\f$ in \f$p\f$ simply provide the exponent vector \f$[a_1,...,a_n]\f$ and the coefficient \f$c\f$. \n
97 For example,
98 
99  p.insert({0,1,4},7);
100 
101 inserts the monomial \f$7x_2x_3^4\f$ in \f$p\f$. \n
102 If we further write:
103 
104  p.insert({1,1,2},-8);
105 
106 then \f$p\f$ becomes \f$-8x_1x_2x_3^2+7x_2x_3^4\f$.\n
107 We can verify that by printing it to the console:
108 
109  std::cout << p << "\n";
110 
111 When inserting monomials in a polynomial, it is the user's responsibility to ensure that:
112 - all exponent vectors have the same length (number of variables) and are distinct
113 - the scalar coefficients are never \f$0\f$
114 
115 Polynomials of the same type and number of variables can be added, subtracted and multiplied via binary operators ```+,-,*``` and raised to a nonnegative integer power by ```^```
116 For example:
117 
118  std::cout << (p+(p^2))<< "\n";
119 
120 will print
121 
122 $$-8x_1x_2x_3^2 + 7x_2x_3^4 + 64x_1^2x_2^2x_3^4 - 112x_1x_2^2x_3^6 + 49x_2^2x_3^8$$
123 
124 which is exactly \f$p+p^2\f$.

126 \subsection symbasis Symmetric Basis
127 
128 We can substitute \ref symmp::StandardVariables with \ref symmp::ElementarySymmetricVariables specifying variables with names \f$e_i\f$, degrees \f$|e_i|=i\f$ and no relations.
129 
130 Example: The element \f$q=-1.5e_1e_2\f$ of the graded ring \f$\mathbf R[e_1,...,e_n], |e_i|=i\f$, can be defined as:
131 
132  Poly<double, ElementarySymmetricVariables<>> q({ 2,3 }, -1.5);
133 
134 We used the constructor that takes a single monomial in the form of exponent+vector.
135 
136 For brevity let us use the following two typedefs:
137 
138  typedef Poly<double, StandardVariables<>> x_poly_t;
139  typedef Poly<double, ElementarySymmetricVariables<>> e_poly_t;
140 
141 Now we view \f[\mathbf R[e_1,...,e_n]=\mathbf R[x_1,...,x_n]^{\Sigma_n}\f] with \f$e_i=\sigma_i\f$ being the elementary symmetric polynomials on the \f$x_i\f$.
142 To convert \f$q\f$ from \f$e_i\f$ variables to \f$x_i\f$ variables we use the class \ref symmp::SymmetricBasis :
143 
144  SymmetricBasis<x_poly_t, e_poly_t> SB(2);
145 
146 The ```2``` signifies that we are using two variables \f$x_1,x_2\f$. Then:
147 
148  auto qx=SB(q);
149  std::cout << qx;
150 
151 will set the polynomial ```x_poly_t qx``` to be ```q``` transformed into the \f$x_i\f$ variables (\ref symmp::StandardVariables) and print \f[-1.5x_1^3x_2^5 + -3x_1^4x_2^4 + -1.5x_1^5x_2^3\f]
152 
153 We can perform the conversion the other way as well: given a polynomial on the \f$x_i\f$ variables such as ```qx``` we can use the object same ```SB``` to transform ``qx`` into a polynomial on the \f$e_i\f$ variables :
154 
155  auto qe=SB(qx);
156  std::cout << qe;
157 
158 which will print \f$-1.5e_1e_2\f$. Observe that ```q==qe``` i.e. the two conversion maps are inverses, as expected.
159 
160 \subsection halfidem Twisted Chern Basis
161 
162 We can generate
163 \f[(R[x_1,...,x_n,y_1,...,y_n]/y_i^2=y_i)^{\Sigma_n}\f]
164 
165 by the \f$\alpha_i, c_i, \gamma_{s,t}\f$ (see \ref math).
166 
167 The class \ref symmp::TwistedChernBasis allows us to transform polynomials on \f$x_i,y_i\f$ variables (\ref symmp::HalfIdempotentVariables) into polynomials on the \f$\alpha_i,c_i,\gamma_{s,i}\f$ (\ref symmp::TwistedChernVariables).
168 
169 Example:
170 
171  typedef Poly<int, HalfIdempotentVariables<>> xy_poly_t;
172  typedef Poly<int, TwistedChernVariables<>>> chern_poly_t;
173  xy_poly_t r;
174  r.insert({ 1,0,1,0 }, 2);
175  r.insert({ 0,1,0,1 }, 2);
176  TwistedChernBasis<xy_poly_t, chern_poly_t> TCB(2);
177  std::cout << TCB(r);
178 
179 This sets ```r``` to be \f$x_1y_1+x_2y_2\f$, transforms it into \f$\gamma_{s,j}\f$ variables and prints the result:
180 
181 \f[-2\gamma_{1,1}+2\alpha_1 c_1\f]
182 
183 And indeed:
184 
185 \f[x_1y_1+x_2y_2=-2\gamma_{1,1}+2\alpha_1 c_1\f]
186 
187 If we perform the transformation again we get the original polynomial:
188 
189  std::cout << TCB(TCB(poly));
190 
191 prints \f$x_1y_1+x_2y_2\f$.
192 
193 We note that the argument ```2``` in the constructor of ```TCB``` is half the number of variables \f$x_1,x_2,y_1,y_2\f$.
194 
195 
196 To print the relations amongst \f$\alpha_i,c_i,\gamma_{s,j}\f$ use:
197 
198  print_half_idempotent_relations<xy_poly_t, chern_poly_t>(3);
199 
200 The \f$3\f$ here corresponds to the half the number of variables: \f$x_1,x_2,x_3,y_1,y_2,y_3\f$ and can be replaced by any positive integer.
201 
202 Finally, Demo.cpp contains another function, \ref pontryagin_via_chern. This prints the expressions of the twisted Pontryagin/symplectic classes:
203 
204 \f[\pi_{s,t}=\kappa_{s,t}=\sum_{1\le i_1< \cdots< i_s\le n\\
205 1\le j_1< \cdots< j_t\le n\\
206 i_u\neq j_v}x^2_{i_1}....x^2_{i_s}y_{j_1}\cdots y_{j_t}\f]
207 
208 in terms of the \f$\alpha_i,c_i,\gamma_{s,t}\f$.
209 For example, try running
210 
211  pontryagin_via_chern < xy_poly_t, chern_poly_t>(5);
