Detailed Description

template<typename T, int N = -1>
struct Symmetric_Polynomials::Half_Idempotent_Variables< T, N >

Class for variables \(x_1,...,x_n,y_1,...,y_n\) where \(y_i^2=y_i\).

Parameter N is the number of variables in compile-time; set to -1 if unknown (default). Otherwise n=N.
