Detailed Description

template<typename scalar_t>
class Symmetric_Polynomials::half_idempotent_generators< scalar_t >

The class containing the generators for the fixed points of $$R=\mathbb Q[x_1,...,x_n,y_1,...,y_n]/(y_i^2=y_i)$$ under the \(\Sigma_n\) permutation action.

There are three types of generators (the following is the default order): Idem ( \(y_1+...+y_n\)), Chern_1,...,Chern_n (elementary symmetric on x_1,...,x_n) as well as the TwistedChern_{s,j} indexed on 1<=s<=n and 1<=j<=n-s The latter involve both the x_i and y_i. The order on the TwistedChern is lexicographic on s,j.
