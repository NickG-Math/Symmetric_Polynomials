◆ SYMMP_RUN_LOOP_IN_PARALLEL

#define SYMMP_RUN_LOOP_IN_PARALLEL

Macro that parallelizes certain loops via openMP if SYMMP_USE_OPEN_MP is defined, and does nothing otherwise.

Only used for the loops in print_half_idempotent_relations and pontryagin_via_chern
To configure the number of threads please set the environment variable OMP_NUM_THREADS on the console before running any executable
