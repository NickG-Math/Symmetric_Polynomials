Detailed Description

Contains the methods and classes for solving the following problem: If $$R=\mathbb Z[x_1,...,x_n,y_1,...,y_n]/(y_i^2=y_i)$$ produce minimal algebra generators for the fixed points of \(R\) under the \(\Sigma_n\) action (permuting the \(x_i,y_i\) separately), give an algorithm for writing a fixed point in terms of the generators and an algorithm for producing the relations of those generators.
