Detailed Description

template<typename scalar_t, typename rel_t>
class Symmetric_Polynomials::monomial< scalar_t, rel_t >

Class of monomials in multiple variables with relations.

A monomial \(a=cx_1^{p_1}x_2^{p_2}....x_n^{p_n}\) is defined by its coefficient \(c\) and exponent vector \([p_1,...,p_n]\). The template scalar_t determines the type of the coefficient \(c\). The template rel_t determines the relations that the variables \(x_i\) satisfy.
