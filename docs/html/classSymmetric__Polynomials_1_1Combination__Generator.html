Detailed Description

template<typename T>
class Symmetric_Polynomials::Combination_Generator< T >

Generates all combinations on a number of letters making a number of choices.

Use with a ranged for loop: Ex for (const auto& i:v){ ... } where v is a combinations_generator object. Then i will be a combination Warning: Not thread safe!

TThe data type of our combinations eg std::vector<int>
