Detailed Description

template<typename T>
struct Symmetric_Polynomials::Half_Idempotent_Variables< T,-1 >

Class specifying variables \(x_1,...,x_n,y_1,...,y_n\) where \(y_i^2=y_i\).

The number of variables n is unknown in compile time.
