Detailed Description

template<typename scalar_t, typename rel_t>
class polynomial< scalar_t, rel_t >

Class of polynomials in multiple variables with relations.

A polynomial is just a vector of monomials, ordered in increasing degree The template scalar_t determines the type of the coefficients. The template rel_t determines the relations that the variables x_i satisfy.
