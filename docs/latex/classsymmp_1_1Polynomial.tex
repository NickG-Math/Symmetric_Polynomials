Class for polynomials in multiple variables with relations. 


\begin{DoxyTemplParams}{Template Parameters}

\hline
{\em \+\_\+exp} & The variable/exponent type of the polynomial eg {\ttfamily \mbox{\hyperlink{structsymmp_1_1StandardVariables}{Standard\+Variables}}} or {\ttfamily \mbox{\hyperlink{structsymmp_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} \\
\hline
{\em \+\_\+cnt} & The data storage type (container) of the polynomial. This should be equivalent to {\ttfamily std\+::map} if {\ttfamily \+\_\+ord==1} and {\ttfamily std\+::unordered\+\_\+map} otherwise \\

\hline
{\em \+\_\+arg} & Any extra optional arguments to pass to {\ttfamily \+\_\+cnt} apart from key,value, comparator/hash (the latter two are provided in {\ttfamily \+\_\+exp}). For example an allocator \\

\end{DoxyTemplParams}
\begin{DoxyParagraph}{Requirements from {\ttfamily exp\+\_\+t}\+:}
The exponent must have functionality similar to {\ttfamily \mbox{\hyperlink{structsymmp_1_1StandardVariables}{Standard\+Variables}}}, {\ttfamily \mbox{\hyperlink{structsymmp_1_1ElementarySymmetricVariables}{Elementary\+Symmetric\+Variables}}}, {\ttfamily \mbox{\hyperlink{structsymmp_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} or {\ttfamily \mbox{\hyperlink{structsymmp_1_1TwistedChernVariables}{Twisted\+Chern\+Variables}}} Specifically\+:
\begin{DoxyItemize}
\item It must have a typedef {\ttfamily deg\+\_\+t} that represents the degree type (eg {\ttfamily int, uint64\+\_\+t})
\item It must have basic vector functionality (constructor that takes {\ttfamily int n} and produces exponent of 0\textquotesingle{}s with that number of variables {\ttfamily n}, {\ttfamily operator \mbox{[}\mbox{]}}...)
\item It should have an operator + to be used in the product of monomials (if products need to be used) and an operator -\/ to be used in the division of monomials (if divisions need to be used).
\item If {\ttfamily \+\_\+ord==1} it needs to have a comparator {\ttfamily bool operator$<$(const \+\_\+exp\&) const} and otherwise it needs to have a hash function {\ttfamily size\+\_\+t operator()() const}
\item Optionally, it may have a {\ttfamily deg\+\_\+t degree() const} method that computes the degree of the monomial exponent
\item Optionally, it may a {\ttfamily std\+::string static name(int var,int num\+\_\+var)} method that prints the name of each variable
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily degree} method then the user needs to provide the dimensions of the variables through a {\ttfamily deg\+\_\+t$\ast$} in the constructor
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily name} method then the user needs to provide the names of the variables through a {\ttfamily std\+::string$\ast$} in the constructor 
\end{DoxyItemize}
\end{DoxyParagraph}
