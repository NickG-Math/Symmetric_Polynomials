Class for polynomials in multiple variables with relations. 


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+scl} & The scalar/coefficient type of the polynomial eg {\ttfamily float} or {\ttfamily int64\+\_\+t} \\
\hline
{\em \+\_\+exp} & The variable/exponent type of the polynomial eg {\ttfamily \mbox{\hyperlink{structsymmp_1_1StandardVariables}{Standard\+Variables}}} or {\ttfamily \mbox{\hyperlink{structsymmp_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} \\
\hline
{\em \+\_\+container} & The data storage type of the polynomial. This should be equivalent to {\ttfamily std\+::map} if {\ttfamily \+\_\+ordered==1} and {\ttfamily std\+::unordered\+\_\+map} otherwise \\
\hline
{\em \+\_\+ordered} & This should be 1 if {\ttfamily \+\_\+container} is equivalent to {\ttfamily std\+::map} and 0 if it\textquotesingle{}s equivalent to {\ttfamily std\+::unordered\+\_\+map} \\
\hline
{\em \+\_\+\+Args} & Any extra optional arguments to pass to {\ttfamily \+\_\+container} apart from key,value, comparator/hash. Eg an allocator \\
\hline
\end{DoxyTemplParams}
\begin{DoxyParagraph}{Requirements from {\ttfamily exp\+\_\+t}\+:}
The exponent must have functionality similar to {\ttfamily \mbox{\hyperlink{structsymmp_1_1StandardVariables}{Standard\+Variables}}} or {\ttfamily \mbox{\hyperlink{structsymmp_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} Specifically\+:
\begin{DoxyItemize}
\item It must have a typedef {\ttfamily deg\+\_\+t} that represents the degree type (eg {\ttfamily int, uint64\+\_\+t})
\item It must have basic vector functionality (constructor that takes {\ttfamily int n} and produces exponent of 0\textquotesingle{}s with that number of variables {\ttfamily n}, operator \mbox{[}\mbox{]}...)
\item It should have an operator + to be used in the product of monomials (if products need to be used) and an operator -\/ to be used in the division of monomials (if divisions need to be used).
\item If {\ttfamily \+\_\+ordered==1} it needs to have a {\ttfamily bool operator$<$() const} and otherwise it needs to have a hash function {\ttfamily size\+\_\+t operator()() const}
\item Optionally, it may have a {\ttfamily deg\+\_\+t degree() const} method that computes the degree of the monomial exponent
\item Optionally, it may a {\ttfamily std\+::string static name(int,int)} method that prints the names of the variables (first parameter is the index of the variable, second is the total number of variables).
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily degree} method then the user needs to provide the dimensions of the variables through a {\ttfamily deg\+\_\+t$\ast$} in the constructor
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily name} method then the user needs to provide the names of the variables through a {\ttfamily std\+::string$\ast$} in the constructor 
\end{DoxyItemize}
\end{DoxyParagraph}
