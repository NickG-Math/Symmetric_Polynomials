\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename scalar\+\_\+t, typename rel\+\_\+t$>$\newline
class Symmetric\+\_\+\+Polynomials\+::monomial$<$ scalar\+\_\+t, rel\+\_\+t $>$}

Class of monomials in multiple variables with relations. 

A monomial $a=cx_1^{p_1}x_2^{p_2}....x_n^{p_n}$ is defined by its coefficient $c$ and powers vector $[p_1,...,p_n]$. The template scalar\+\_\+t determines the type of the coefficient $c$. The template rel\+\_\+t determines the relations that the variables $x_i$ satisfy.
