class Symmetric\+\_\+\+Polynomials\+::half\+\_\+idempotent\+\_\+generators$<$ scalar\+\_\+t $>$}

The class containing the generators for the fixed points of \$\$R=\textbackslash{}mathbb Q\mbox{[}x\+\_\+1,...,x\+\_\+n,y\+\_\+1,...,y\+\_\+n\mbox{]}/(y\+\_\+i$^\wedge$2=y\+\_\+i)\$\$ under the $\Sigma_n$ permutation action. 

There are three types of generators (the following is the default order)\+: Idem ( $y_1+...+y_n$), Chern\+\_\+1,...,Chern\+\_\+n (elementary symmetric on x\+\_\+1,...,x\+\_\+n) as well as the Twisted\+Chern\+\_\+\{s,j\} indexed on 1$<$=s$<$=n and 1$<$=j$<$=n-\/s The latter involve both the x\+\_\+i and y\+\_\+i. The order on the Twisted\+Chern is lexicographic on s,j.
