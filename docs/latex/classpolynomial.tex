\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename scalar\+\_\+t, typename rel\+\_\+t$>$\newline
class polynomial$<$ scalar\+\_\+t, rel\+\_\+t $>$}

Class of polynomials in multiple variables with relations. 

A polynomial is just a vector of monomials, ordered in increasing degree The template scalar\+\_\+t determines the type of the coefficients. The template rel\+\_\+t determines the relations that the variables x\+\_\+i satisfy.
