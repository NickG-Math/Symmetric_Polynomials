\hypertarget{use_sym}{}\doxysection{Symmetric polynomials}\label{use_sym}
Let $R=\mathbb Q[x_1,...,x_n]$; there is an obvious action on $R$ by the symmetric group $\Sigma_n$ and the fixed points $R^{\Sigma_n}$ i.\+e. the symmetric polynomials, form a polynomial algebra on the elementary symmetric polynomials\+: \$\$R$^\wedge$\{\textbackslash{}\+Sigma\+\_\+n\}=\textbackslash{}mathbb Q\mbox{[}\textbackslash{}sigma\+\_\+1,...,\textbackslash{}sigma\+\_\+n\mbox{]}\$\$ where \$\$\textbackslash{}sigma\+\_\+k(x\+\_\+1,...,x\+\_\+n)=\textbackslash{}sum\+\_\+\{1\textbackslash{}le i\+\_\+1$<$ \textbackslash{}cdots$<$ i\+\_\+k\textbackslash{}le n\}x\+\_\+\{i\+\_\+1\}....x\+\_\+\{i\+\_\+k\}\$\$ Furthermore, there is a simple algorithm for writing every symmetric polynomial on the $x_i$ as a polynomial on the $\sigma_i$. This library implements that algorithm.\hypertarget{math_symr}{}\doxysection{Symmetric polynomials with \textquotesingle{}half idempotent\textquotesingle{} relations}\label{math_symr}
Let $R=\mathbb Q[x_1,...,x_n,y_1,...,y_n]/(y_i^2=y_i)$; there is an obvious action on $R$ by the symmetric group $\Sigma_n$ permuting the $x_i$ and $y_i$ variables separately. A minimal description of the $R^{\Sigma_n}$ is now more difficult\+: \$\$R$^\wedge$\{\textbackslash{}\+Sigma\+\_\+n\}=\textbackslash{}frac\{\textbackslash{}mathbb Q\mbox{[}\textbackslash{}alpha,c\+\_\+i,\textbackslash{}gamma\+\_\+\{s,t\}\mbox{]}\}\{\textbackslash{}sim\}\$\$ where \$\$\textbackslash{}alpha=y\+\_\+1+\textbackslash{}cdots+y\+\_\+n\$\$ \$\$c\+\_\+k=\textbackslash{}sum\+\_\+\{1\textbackslash{}le i\+\_\+1$<$ \textbackslash{}cdots$<$ i\+\_\+k\textbackslash{}le n\}x\+\_\+\{i\+\_\+1\}....x\+\_\+\{i\+\_\+k\}\$\$ \$\$\textbackslash{}gamma\+\_\+\{s,t\}=\textbackslash{}sum\+\_\+\{1\textbackslash{}le i\+\_\+1$<$ \textbackslash{}cdots$<$ i\+\_\+s\textbackslash{}le n\textbackslash{} 1\textbackslash{}le j\+\_\+1$<$ \textbackslash{}cdots$<$ j\+\_\+t\textbackslash{}le n\textbackslash{} i\+\_\+u\textbackslash{}neq j\+\_\+v\}x\+\_\+\{i\+\_\+1\}....x\+\_\+\{i\+\_\+s\}y\+\_\+\{j\+\_\+1\}\textbackslash{}cdots y\+\_\+\{j\+\_\+t\}\$\$ where $1\le i\le n$ and $1\le s\le n$ and $1\le t\le n-s$.

The relations $\sim$ are\+:

\$\$\textbackslash{}alpha$^\wedge$\{n+1\}=\textbackslash{}alpha$^\wedge$n+\textbackslash{}cdots\$\$ \$\$\textbackslash{}alpha$^\wedge$s\textbackslash{}gamma\+\_\+\{s,i\}=c\+\_\+s\textbackslash{}alpha$^\wedge$\{i+s\}+\textbackslash{}cdots\$\$ \$\$\textbackslash{}gamma\+\_\+\{s,i\}\textbackslash{}gamma\+\_\+\{t,j\}=c\+\_\+t\textbackslash{}gamma\+\_\+\{s,\textbackslash{}min(i+j,n)\}+\textbackslash{}cdots\$\$ where $s\le t\le s+i$ for the last relation.

The first relation can be explicitly computed as\+: \$\$ \textbackslash{}alpha(\textbackslash{}alpha-\/1)\textbackslash{}cdots (\textbackslash{}alpha-\/n)=0\$\$

This library implements an algorithm that can write every element in $R^{\Sigma_n}$ as a polynomial on the generators $\alpha,c_i,\gamma_{s,t}$ and further produce every relation explicitly. The class $\alpha$ is referred to as \char`\"{}idempotent class\char`\"{} (since it\textquotesingle{}s made out of idempotents), the classes $c_i$ are referred to as \char`\"{}\+Chern classes\char`\"{} and the classes $\gamma_{s,t}$ are the \char`\"{}twisted Chern classes\char`\"{}.
