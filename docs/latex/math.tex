Let $R=\mathbb Z[x_1,...,x_n]$; there is an obvious action on $R$ by the symmetric group $\Sigma_n$ and the fixed points $R^{\Sigma_n}$ i.\+e. the symmetric polynomials, form a polynomial algebra on the elementary symmetric polynomials\+: \$\$R$^\wedge$\{\textbackslash{}\+Sigma\+\_\+n\}=\textbackslash{}mathbb Z\mbox{[}\textbackslash{}sigma\+\_\+1,...,\textbackslash{}sigma\+\_\+n\mbox{]}\$\$ where \$\$\textbackslash{}sigma\+\_\+k(x\+\_\+1,...,x\+\_\+n)=\textbackslash{}sum\+\_\+\{1\textbackslash{}le i\+\_\+1$<$ \textbackslash{}cdots$<$ i\+\_\+k\textbackslash{}le n\}x\+\_\+\{i\+\_\+1\}....x\+\_\+\{i\+\_\+k\}\$\$ Furthermore, there is a simple algorithm for writing every symmetric polynomial on the $x_i$ as a polynomial on the $\sigma_i$. Let $R=\mathbb Z[x_1,...,x_n,y_1,...,y_n]/(y_i^2=y_i)$; there is an obvious action on $R$ by the symmetric group $\Sigma_n$ permuting the $x_i$ and $y_i$ variables separately. A minimal description of the $R^{\Sigma_n}$ is now more difficult\+: \$\$R$^\wedge$\{\textbackslash{}\+Sigma\+\_\+n\}=\textbackslash{}frac\{\textbackslash{}mathbb Z\mbox{[}\textbackslash{}gamma\+\_\+\{s,i\}\mbox{]}\}\{\textbackslash{}gamma\+\_\+\{s,i\}\textbackslash{}gamma\+\_\+\{t,j\}=\{\textbackslash{}min(i+j+s,n)-\/t\}\textbackslash{}choose \{j\}\textbackslash{}gamma\+\_\+\{t,0\}\textbackslash{}gamma\+\_\+\{s,\textbackslash{}min(i+j,n)\}+\textbackslash{}cdots\}\$\$ where the \char`\"{}twisted Chern classes\char`\"{} are\+: \$\$\textbackslash{}gamma\+\_\+\{s,i\}=\textbackslash{}sum\+\_\+\{1\textbackslash{}le j\+\_\+1$<$ \textbackslash{}cdots$<$ j\+\_\+s\textbackslash{}le n, 1\textbackslash{}le k\+\_\+1$<$ \textbackslash{}cdots$<$ k\+\_\+i\textbackslash{}le n\textbackslash{} j\+\_\+u\textbackslash{}neq k\+\_\+v\}x\+\_\+\{j\+\_\+1\}....x\+\_\+\{j\+\_\+s\}y\+\_\+\{k\+\_\+1\}\textbackslash{}cdots y\+\_\+\{k\+\_\+i\}\$\$ for $0\le s\le n$ and $0\le i\le n-s$. The relations require the indices $s,i,t,j$ to satisfy $s\le t\le s+i$ and $i,j>0$.

For convenience we set\+: \$\$\textbackslash{}alpha\+\_\+i=\textbackslash{}gamma\+\_\+\{0,i\}=\textbackslash{}sigma\+\_\+i(y\+\_\+1,...,y\+\_\+n)\$\$ \$\$c\+\_\+s=\textbackslash{}gamma\+\_\+\{s,0\}=\textbackslash{}sigma\+\_\+s(x\+\_\+1,...,x\+\_\+n)\$\$ Over $\mathbb Q$, the $\alpha_1^i$ can generate all $\alpha_i$ via\+: \$\$\textbackslash{}alpha\+\_\+i=\textbackslash{}frac\{\textbackslash{}alpha(\textbackslash{}alpha-\/1)\textbackslash{}cdots(\textbackslash{}alpha-\/i+1)\}\{i!\}\$\$ however for speed+numerical stability we prefer to use all $\alpha_i$ and have $\mathbb Z$ coefficients in our relations.

This library implements an algorithm that can write every element in $R^{\Sigma_n}$ as a polynomial on the generators $\alpha_i,c_i,\gamma_{s,i}$ and further produce every relation explicitly.
