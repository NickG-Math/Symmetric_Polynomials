\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename container\+\_\+1\+\_\+t, typename container\+\_\+2\+\_\+t$>$\newline
class Symmetric\+\_\+\+Polynomials\+::\+Polynomial\+\_\+\+Basis$<$ T, container\+\_\+1\+\_\+t, container\+\_\+2\+\_\+t $>$}

Factory class that provides the general interface of a generating basis for a subring of a polynomial ring. Example implementations are \mbox{\hyperlink{classSymmetric__Polynomials_1_1Symmetric__Basis}{Symmetric\+\_\+\+Basis}} and \mbox{\hyperlink{classSymmetric__Polynomials_1_1Half__Idempotent__Basis}{Half\+\_\+\+Idempotent\+\_\+\+Basis}}. 

Use by inheriting and setting T be the inheriting class (C\+R\+TP). Then construct \+\_\+generators (an optionally generator\+\_\+names and generator\+\_\+dimensions) in the inheriting class.

The inheriting class T must implement a method find\+\_\+exponent with singature\+: \mbox{\hyperlink{classSymmetric__Polynomials_1_1Polynomial_aa465567e5f113d3366dc71ff32af55b1}{Polynomial$<$container\+\_\+2\+\_\+t$>$\+::exp\+\_\+t}} find\+\_\+exponent(const Polynomial$<$container\+\_\+1\+\_\+t$>$\+::exp\+\_\+t\&);

container\+\_\+1\+\_\+t is the container type of the original variables (of the polynomial ring)

container\+\_\+2\+\_\+t is the container type of the new variables that are the basis elements (of the subring)
