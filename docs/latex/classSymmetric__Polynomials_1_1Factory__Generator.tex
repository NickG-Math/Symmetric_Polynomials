Prototype for coroutine-\/like iterators that generate elements such as interpolating vectors, permutations, combinations... 

Inherit from this class and define a method update() to get a const iterator. ~\newline
 You will also need begin() and \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a703e81ab4d6c144c8b926b4c0121ea3e}{end()}} methods constructing such iterators; \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a703e81ab4d6c144c8b926b4c0121ea3e}{end()}} should always be defined by calling the factory \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a703e81ab4d6c144c8b926b4c0121ea3e}{end()}}.~\newline
 Example implementations\+: {\ttfamily \mbox{\hyperlink{classSymmetric__Polynomials_1_1Combination__Generator}{Combination\+\_\+\+Generator}}} and {\ttfamily \mbox{\hyperlink{classSymmetric__Polynomials_1_1Permutation__Generator}{Permutation\+\_\+\+Generator}}} \begin{DoxyAttention}{Attention}
