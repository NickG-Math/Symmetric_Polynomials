\subsubsection*{template$<$typename specialization, typename gen\+\_\+t$>$\newline
class Symmetric\+\_\+\+Polynomials\+::\+Factory\+\_\+\+Generator$<$ specialization, gen\+\_\+t $>$}

Prototype for classes providing iterators that generate elements such as interpolating vectors, permutations, combinations and so on. 

Inherit from this class and define a method update() to get a const iterator. You will also need begin() and \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a4c5bb7b8ec49b21e6aa3ec09e4d435e7}{end()}} methods constructing such iterators; \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a4c5bb7b8ec49b21e6aa3ec09e4d435e7}{end()}} should always be defined by calling the factory \mbox{\hyperlink{classSymmetric__Polynomials_1_1Factory__Generator_a4c5bb7b8ec49b21e6aa3ec09e4d435e7}{end()}}. Warning\+: Probably not thread-\/safe (depends on child\textquotesingle{}s update() method).
