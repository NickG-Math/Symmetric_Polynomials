Factory class that provides the general interface of a generating basis for a subring of a polynomial ring. 

Inherit from this class and construct data member \mbox{\hyperlink{classSymmetric__Polynomials_1_1PolynomialBasis_a82a0b79e96aec4dde7f31917dd46fc44}{\+\_\+generators}} through the child class (and optionally \mbox{\hyperlink{classSymmetric__Polynomials_1_1PolynomialBasis_a4867a7fd4fad1c2ce7a65656ee5a0bd7}{generator\+\_\+names}} and \mbox{\hyperlink{classSymmetric__Polynomials_1_1PolynomialBasis_afba7b5d8ad929bf149ad3fc0b35d06ec}{generator\+\_\+dimensions}}). ~\newline
 The child class must also have a method {\ttfamily find\+\_\+exponent} with singature\+:
