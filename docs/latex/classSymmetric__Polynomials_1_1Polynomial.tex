\begin{DoxyParagraph}{Requirements from {\ttfamily exp\+\_\+t}\+:}
The exponent must have functionality similar to {\ttfamily \mbox{\hyperlink{structSymmetric__Polynomials_1_1StandardVariables}{Standard\+Variables}}} or {\ttfamily \mbox{\hyperlink{structSymmetric__Polynomials_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} Specifically\+:
\begin{DoxyItemize}
\item It must have a typedef {\ttfamily deg\+\_\+t} that represents the degree type (eg {\ttfamily int},uint64\+\_\+t)
\item It must have basic vector functionality (constructor that takes {\ttfamily int} {\ttfamily n} and produces exponent of 0\textquotesingle{}s with that number of variables {\ttfamily n}, operator \mbox{[}\mbox{]}...)
\item It should have an operator + to be used in the product of monomials (if products need to be used) and an operator -\/ to be used in the division of monomials (if divisions need to be used).
\item If {\ttfamily container\+\_\+is\+\_\+ordered==1} it needs to have a {\ttfamily bool} {\ttfamily operator$<$()} {\ttfamily const} and otherwise it needs to have a hash function {\ttfamily size\+\_\+t} {\ttfamily operator()()} {\ttfamily const} 
\item Optionally, it may have a {\ttfamily deg\+\_\+t} {\ttfamily degree()} {\ttfamily const} method that computes the degree of the monomial exponent
\item Optionally, it may a {\ttfamily std\+::string} {\ttfamily static} {\ttfamily name(int,int)} method that prints the names of the variables (first parameter is the index of the variable, second is the total number of variables).
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily degree} method then the user needs to provide the dimensions of the variables through a {\ttfamily deg\+\_\+t$\ast$} in the constructor
\item If {\ttfamily exp\+\_\+t} does not have a {\ttfamily name} method then the user needs to provide the names of the variables through a {\ttfamily std\+::string$\ast$} in the constructor 
\end{DoxyItemize}
\end{DoxyParagraph}
