\subsubsection*{template$<$class \+\_\+scl, class \+\_\+exp, template$<$ class... $>$ class \+\_\+cnt, bool \+\_\+ord, class ... \+\_\+arg$>$\newline
class symmp\+::\+Default\+Container$<$ \+\_\+scl, \+\_\+exp, \+\_\+cnt, \+\_\+ord, \+\_\+arg $>$}

The default ordered/unordered monomial container. 

Monomials are stored as key-\/value pairs where each key consists of degree+exponent and the value is the coefficient of the monomial. \begin{DoxyNote}{Note}
We store the degree for improved performance. 
\end{DoxyNote}

\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+scl} & The scalar/coefficient type of the monomials eg {\ttfamily float} or {\ttfamily int64\+\_\+t} \\
\hline
{\em \+\_\+exp} & The variable/exponent type of the monomials eg {\ttfamily \mbox{\hyperlink{structsymmp_1_1StandardVariables}{Standard\+Variables}}} or {\ttfamily \mbox{\hyperlink{structsymmp_1_1HalfIdempotentVariables}{Half\+Idempotent\+Variables}}} \\
\hline
{\em \+\_\+cnt} & The underlying container\+: this should be equivalent to {\ttfamily std\+::map} if {\ttfamily \+\_\+ord==1} and {\ttfamily std\+::unordered\+\_\+map} otherwise \\
\hline
{\em \+\_\+ord} & This should be 1 if {\ttfamily \+\_\+cnt} is equivalent to {\ttfamily std\+::map} and 0 if it\textquotesingle{}s equivalent to {\ttfamily std\+::unordered\+\_\+map} \\
\hline
{\em \+\_\+arg} & Any extra optional arguments to pass to {\ttfamily \+\_\+cnt} apart from key,value, comparator/hash (the latter two are provided in {\ttfamily \+\_\+exp}) eg an allocator \\
\hline
\end{DoxyTemplParams}
